#] Discord permissions descriptions

Discord's descriptions for their permissions are a little lack-luster. Let's fix that.
Permission Description Value
CREATE_INSTANT_INVITE Allows anybody with this permission to create an invite to the guild. Does not affect the "Invite your friends" button at the top of the channel though. 0x00000001
KICK_MEMBERS Allows anybody with this permission to kick members. 0x00000002
BAN_MEMBERS Allows anybody with this permission to ban members. Bots cannot be banned. 0x00000004
ADMINISTRATOR Basically gives all permissions in the guild except for managing above roles. Also disables slowmode for this role. 0x00000008
MANAGE_CHANNELS Allows anybody with this permission to manage any text channel, news channel, voice channel, announcement channel or category channel that they can see. Also disables slowmode for this role. 0x00000010
MANAGE_GUILD Allows anybody with this permission to manage the guild name, region, banner, AFK voice channels, system messages and channels, notification settings and more. 0x00000020
ADD_REACTIONS Allows anybody with this permission to add new reactions to a message. Existing reactions can always be reacted to. 0x00000040
VIEW_AUDIT_LOG Allows anybody with this permission to view the audit log. That's it. 0x00000080
PRIORITY_SPEAKER Allows anybody with this permission to have priority over the others in a voice channel. 0x00000100
STREAM Allows anybody with this permission to stream video in a voice channel. Bots cannot stream video. 0x00000200
READ_MESSAGES Allows anybody with this permission to read messages in a text channel, announcement channel or news channel. 0x00000400
SEND_MESSAGES Allows anybody with this permission to send messages in a text channel, announcement channel or news channel. Previously sent messages cannot be edited either. 0x00000800
SEND_TTS_MESSAGES Allows anybody with this permission to use the /tts command when sending a message to automatically speak out the message to all users that enabled TTS and are currently viewing that channel. TTS is not supported on Linux. 0x00001000
MANAGE_MESSAGES Allows anybody with this permission to delete, pin, or edit reactions of any message. 0x00002000
EMBED_LINKS Allows anybody with this permission to embed links such as YouTube videos or tweets. If the message only contains a link to an image or video file, then the link is hidden and the image is shown. The embedded image or video does NOT count as an attachment when working with the API. 0x00004000
ATTACH_FILES Allows anybody with this permission to upload files. The attached file does NOT count as an embed, and links to image or video files does NOT count as attaching a file. 0x00008000
READ_MESSAGE_HISTORY Allows anybody with this permission to scroll back in the message history from when they started viewing it. 0x00010000
MENTION_EVERYONE Allows anybody with this permission to use @everyone @here or mention any existing role, even if that role explicitly states to not let everyone mention it. 0x00020000
EXTERNAL_EMOJIS Allows anybody with this permission to use emojis not available in the guild as a reaction or in the message content. Bots can use external emojis if they have this permission and can access the emoji. 0x00040000
VIEW_GUILD_INSIGHTS Allows anybody with this permission to view the guild insights of a community guild. This permission is not available for non community guilds. 0x00080000
CONNECT Allows anybody with this permission to connect to a voice channel. This does not guarentee the ability to speak. 0x00100000
SPEAK Allows anybody with this permission to speak in a voice channel. This does not mean that they can connect. 0x00200000
